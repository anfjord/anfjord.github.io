<Array

An array is a collection of objects that can be given a variable name and stored in your program. The object stored in an array can include strings, integer and hashes. Using arrays is an efficient way to store a list and and being able to access item in the list. Arrays are integer-indexed which means that you access items in the array by using the items idea number. The first item in an array has the index number 0, the second one has the index number 1 and so on.

For example: if I have this array my_ names = [“Anna”, “Maria”, “Johanna”, “Marageta”] and want to access Margareta I will use the index number 3, my_names [3]

Hashes

A hash is a collection of items that can be stored in pairs called keys and values. The key must be unique and cannot be repeated in the hash. You can always add, remove, or replace items in the hash Lets take an example:

Lets take an example:

my_hash = {“apples” => “5 USD”, “mangos” => " 6 USD”, “oranges” => “7 USD”}

To access the value you use the key. If you want the price of apples, you can access it by typing my_hash[“apples”}.
