#Map it out

Enumerable

Enumerable is a set of methods bundled together that can manipulate collections. Using Enumerable you don't have to write a bunch of methods like search, sort and manipulate collection many different time, just once! All of the methods in Enumerable are based on the each method. It means that the class needs to have its own #each method that Enumerable can use in order to access all the bundled methods.

#map

he enumerable #map will take an object and run a block of code for each element in the object and collect each element and put it in an array. Map is sometimes called collect. The main difference from using each is that map returns the transformed collection of elements.

The syntax for #map looks like this: object.map {| element | block of code}.

Lets take an example using a hash.

This is a collection of my Swedish stocks have moved south today and I wanna add the message “Sell!” to each one that went down.

my_Swedish_stocks = {:Volvo => "down", :Ericsson => "down", :Scania => "down"}

stocks_to_sell = my_Swedish_stocks.map {|name,movement| {name => movement + " Sell!"}}

It will return:

=> [{:Volvo=>"down Sell!"}, {:Ericsson=>"down Sell!"}, {:Scania=>"down Sell!"}]
